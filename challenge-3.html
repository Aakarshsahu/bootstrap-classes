Alerts

Alerts are created with the .alert class, followed by a contextual colorclasses:

Success! This alert box could indicate a successful orpositive action.
Info! This alert box could indicate a neutral informativechange or action.
Warning! This alert box could indicate a warning thatmight need attention.
Danger! This alert box could indicate a dangerous orpotentially negative action.
Primary! Indicates an important action.
Secondary! Indicates a slightly less important action.
Dark! Dark grey alert.
Light! Light grey alert.
